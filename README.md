## **Migrations**

---

1. cd app/CleanArch/CleanArch.Infra.Data

2. dotnet ef --startup-project ../CleanArch.MVC/ migrations add Initial

#### **Update Database**

1. cd app/CleanArch/CleanArch.Infra.Data

2. dotnet ef --startup-project ../CleanArch.MVC/ database update
